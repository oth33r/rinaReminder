# Telegram pill reminder bot

Бот на Python для Telegram, который:

- хранит несколько записей с временем и описанием;
- управляется через кнопки в чате;
- напоминает выпить таблетку в заданное время;
- повторяет напоминание каждые 10 минут, пока пользователь не нажмет кнопку `Выпила`;
- умеет по кнопке отправлять комплиментик с сердечком;
- после подтверждения отправляет милое сообщение.

## Запуск

1. Создай бота через `@BotFather` и получи токен.
2. Создай файл `.env` рядом с `bot.py` и вставь:

```env
TELEGRAM_BOT_TOKEN=твой_токен
```

3. Установи зависимости:

```bash
python -m pip install -r requirements.txt
```

4. Запусти бота:

```bash
python bot.py
```

## Кнопки

- `Добавить запись` - пошагово создает новое напоминание;
- `Мои записи` - показывает все записи и кнопки `Изменить` и `Удалить`;
- `Комплиментик ❤️` - отправляет милое сообщение;
- `Удалить все` - очищает все записи;
- `Отмена` - сбрасывает текущее действие.

## Команды

- `/start` - показать приветствие и меню;
- `/help` - показать помощь.
